Краткое руководство пользователя программой для генерации речеподобных помех.



I. Сведения о виджетах


Главное окно содержит:

1. Графический фрейм "Параметры помехового сигнала", который включает в себя:
       - радиокнопки определения частоты дискретизации
       (предназначены для определения частоты дискретизации помехового сигнала);
       - радиокнопки определения формата кодирования
       (предназначены для определения формата кодирования помехового сигнала);
       - радиокнопки определения числа каналов
       (предназначены для определения числа каналов помехового сигнала);
       - однострочное текстовое поле определения длительности базового звукового фрагмента
       (предназначено для ввода пользователем длительности базового звукового фрагмента,
       из которых формируется помеховый сигнал);
       - однострочное текстовое поле определения длительности помехового сигнала
       (предназначено для ввода пользователем продолжительности помехового сигнала);

2. Кнопка "Открыть меню записи голоса"
   (при нажатии на кнопку открывается окно "Меню записи голоса")
   
3. Графический фрейм "Меню генерации помехи", который включает в себя:
       - однострочное текстовое поле определения числа используемых для генерации аудиофайлов
       (предназначено для ввода пользователем числа файлов, используемых для генерации помехи);
       - кнопка "Прикрепить аудиофайлы..."
       (при нажатии на кнопку открывается окно "Получение данных о файлах";
       - индикатор прикрепления аудиофайлов
       (предназначен для индикации статуса прикрепления аудиофалов и вывода сообщения о статусе прикрепления);
       - кнопка "Сгенерировать помеху"
       (предназначена для запуска процесса обработки аудиофайлов).

4. Кнопка строки меню "Справка"
   (при нажатии на кнопку открывается краткое руководство пользователя)

5. Эмблема организации


Окно "Меню записи голоса" (открывается нажатием кнопки "Открыть меню записи голоса") содержит:

1. Радиокнопки определения частоты дискретизации для процесса записи;
2. Радиокнопки определения формата кодирования для процесса записи;
3. Радиокнопки определения числа каналов для процесса записи;
4. Однострочное текстовое поле определения продолжительности записи;
5. Однострочное текстовое поле ввода индекса аудиоустройства, используемого для записи
   (предназначено для указания аудиоустройства, которое будет использовано для записи);
6. Кнопка "Получить список доступных аудиоустройств"
   (при нажатии на кнопку открывается окно "Информация о подключенных аудиоустройствах";
7. Кнопка "Начать запись голоса"
   (при нажатии на кнопку открывается диалоговое окно с информацией о начале записи;
   при нажатии кнокпи "ОК" в диалоговом окне звучит короткий звуковой сигнал начала записи
   и открывается осцилограмма, отображающая процесс записи голоса в режиме реального времени);
 
Окно "Получение данных о файлах" содержит:

1. Однострочные текстовые поля определения путей к аудиофайлам
   (предназначено для ввода пользователем абсолютных путей к аудиофайлам)
2. Кнопки "Указать путь"
   (при нажатии на кнопку открывается стандартное диалоговое окно для указания пути к аудиофайлу);
3. Кнопка "Прикрепить"
   (при нажатии на кнопку выбранные аудиофайлы поступают в память программы,
   индикатор прикрепления аудиофайлов изменяет свой цвет с красного на зелёный,
   выводится сообщение о готовности файлов к обработке)



II. Порядок действий для генерации помехового сигнала


1. В графическом фрейме "Параметры помехового сигнала" с помощью радиокнопок задайте:
   - частоту дискретизации;
   - формат кодирования;
   - число каналов;
   - длительность базового фрагмента (в миллисекундах), 
     который будет использован для генерации помехового сигнала.
   - продолжительность помехового сигнала в секундах.
2. В графическом фрейме "Меню генерации помехи" укажите число аудиофайлов, используемых для генерации помехи.
   (для генерации одноголосной помехи введите "1", 
   для генерации помехи типа "речевой хор" введите число, равное числу требуемых голосов).
3. Нажмите кнопку "Прикрепить аудиофайлы"
   (В появившемся окне, нажимая на кнопки "Указать путь", 
   укажите пути к аудиофайлам и нажмите кнопку "ПРИКРЕПИТЬ).
6. Убедитесь, что индикатор прикрепления аудиофайлов в графическом фрейме "Меню генерации помехи"
   изменил свой цвет на зелёный и в нём появилось сообщение "Файлы готовы").
7. Нажмите кнопку "Сгенерировать помеху".
8. В открывшемся диалоговом окне укажите путь для сохранения .wav файла с помеховым сигналом.



III. Порядок действий для записи голоса


1. Нажмите на кнопку "Открыть меню записи голоса"
2. В появившемся окне "Меню записи голоса" задайте:
   - частоту дискретизации;
   - формат кодирования;
   - число каналов;
   - продолжительность помехового сигнала в секундах;
   - индекс аудиоустройства, используемого для записи
     (получить информацию о подключенных аудиоустройствах
     можно, нажав на кнопку "Получить список доступных аудиоустройств")
3. Нажмите на кнопку "Начать запись голос"
4. В появившемся диалоговом окне со служебной информацией нажмите "ОК"
5. Начинайте говорить после звукового сигнала
   (процесс записи сопровождается осцилограммой голоса
   в режиме реального времени).
6. В открывшемся диалоговом окне укажите путь для сохранения .wav файла записи
   (записанный файл можно использовать для получения помехового сигнала,
   см. пункт II)
